## 문제 2.

- Vue Router는 두 가지 라우팅 모드를 지원하는데, 하나는 `#(해쉬)` 모드이고 다른 하나는 `History` 모드
- `vue-router`의 기본 모드는 hash mode이다. 
  URL 해시를 사용하여 전체 URL을 시뮬레이트하므로 URL이 변경될 때 페이지가 다시 로드되지 않는다.
